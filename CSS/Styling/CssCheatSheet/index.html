CSS Selectors and Specificity

All CSS Selectors

Selector Name Description
div Tag Selector Select the whole tag with specific name.
.class Class Selector Select the element with the given class.
#id Id Selector Select the element or tag with given id name.

Specificity Of Selectors

Rank Name Description
Inline Style style=" " Highest specificity because presented at the last.
#id Id Selector Important because each element has a unique ID
.class Class Selector There are multiple elements having same class
div Tag Selector There are multiple or countless div elements in a page.
